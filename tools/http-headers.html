HTTP Headers 分析

输入网址，查看其 HTTP 响应头信息，并检测关键安全头的配置情况。

HTTP 响应头是服务器在返回网页内容时发送的一系列元数据，包含了服务器类型、内容编码、缓存策略和安全配置等重要信息。常见的安全相关响应头包括：Content-Security-Policy（CSP）用于限制页面可以加载的资源来源，防止 XSS 攻击；Strict-Transport-Security（HSTS）强制浏览器使用 HTTPS 连接；X-Frame-Options 控制页面是否可以被嵌入 iframe，防止点击劫持；X-Content-Type-Options 防止浏览器进行 MIME 类型嗅探。通过本工具，你可以查看任意网站的 HTTP 响应头信息，检测安全头配置是否完善，提升网站安全等级。
